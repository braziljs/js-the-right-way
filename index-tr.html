BAŞLARKEN

HAKKINDA

1995 yılında Netscape tarafından Netscape Navigotor 2.0 için yaratılan HTML eklentisi olan Javascriptin From doğrulama ve HTML dosyalarında değişiklikler yapmak temel özelliklerindendir. Bu günlerde bu isim ün kazanmadan önce, Javascript "Mocha" olarak adlandırılıyordu, sonra "LiveScript" dendi ve son olarak Sun Microsystems tarafından yayınlandıktan sonra bu günkü ismini aldı. İsim benzerliği yüzünden insanlar Java ve JavaScript'i karıştırıyor. İkiside Sözlük yapısına uygun olmasına rağmen ikiside aynı dil değildir. C, C# ve Java'dan farklı olarak JavaScript yorumlanabilir bir dildir. Bu şu anlama gelir bir yorumlayıcıya ihtiyacı vardır. JavaScript'te bu yorumlayıcı tarayıcıdır.

GEÇERLİ VERSİYON

JavaScript standardı ECMAScript'tir. 2012 yılından itibaren tüm modern tarayıcılar ECMAScript 5.1'i tamamen deteklemeye başlamıştır. Daha eski tarayıcılar en son ECMAScript 3'ü desteklemektedir. Çalışması, altıncı güncellemeden sonra standartlaşmıştır.

Mozilla Geliştirici ağı adresinden en yeni JavaScript haberlerine ve referanslarını bulabilirsiz.

THE DOM

The Document Object Model (DOM), HTML ve XML dosyaları için bir kütüphanedir. JavaScript dökümanların yapısal gösterimlerini sağlar; dili kullanarak sizin içerik ve görsel sunum ile ilgili değişikilik yapmanıza olanak tanır. Daha fazlası için Mozilla Developer Network - DOM.

JS KOD STİLİ

YÖNTEM

Her dil gibi JavaScript'inde bir çok kod stil rehberi vardır. Belki en çok tavsite edilen ve kullanılan rehber Google Code Style Guide for JavaScript budur fakat biz size bunu okumanızı öneriyoruz Idiomatic.js.

OKUNABİLİRLİĞİ ARTTIRMA

Bu günlerde JavaScript kodunun okunulabilirliğini düzenlemek için kullanılan en iyi araç JSHinttir. Ne zaman kod stilinizi ve yazılım mimarinizi doğrulamanız gerekirse Lint tool'u kullanmanızı öneririz.

EN İYİ BÖLÜM

Nesne Tabanlı

JavaScript güçlü bir nesene tabanlı programlama kapasitesine sahiptir, buna rağmen başka diller ve JavaScript karşılaştırıldığında nesne tabanlı programlama farklılıklarına bağlı olarak bazı tartışmalar oluşmuştur.

Kaynak: Introduction to Object-Oriented JavaScript

İSİMSİZ FONKSİYONLAR

İsimsiz fonksiyonları çalışırken dinamikliği bildiren fonksiyonlardır. İsimsiz fonskyion olarak adlandırılırlar çünkü normal fonksiyonlar gibi aynı yolla isim verilmemiştir.

Kaynak: Javascript anonymous functions

FONKSİYONLAR BİRİNCİ SINIF NESNELERDİR

Fonksiyonlar javaScript içinde birinci sınıf nesnelerdir. Bu anlamı, javaScript fonksiyonları düzenli nesnelerin her işi yapabilen bir türüdür.

Kaynak: Functions are first class objects in JavaScript

ESNEK YAZIM BİÇİMİ

Birçok UI geliştirici için JS, betik ve/veya yorumlanabilir dilden aldıkları ilk tattır. Bu geliştiriciler için, konvansiyonel kodlama ve konseptler ikinci planda tutulabilir. Yanı sıra, modern web uygulamalarına olan talep patlaması, back-end geliştiricilerin kolları sıvayarak, bu dünyaya adım atmalarını gerektirdi. Bu geliştiricilerin çoğu ise, C# ve Java tabanlı olduklarından, özgür ve esnek bir şekilde yazılım geliştirmeye açık değillerdi.

Kaynak: Understanding Loose Typing in JavaScript

YAZILIM OMURGALARI

İSTEMCİ TARAFI

  • jQuery

    • jQuery hızlı, küçük ve özellik açısından zengin bir JavaScript kütüphanesidir. John Resig tarafından tasarlanmıştır.

  • Backbone.js

    • @jashkenas tarafından tasarlanan çok popüler bir istemci tarafı JavaScript kütüphanesidir.

  • Ember.js

    • jQuery ve Ruby on Rails geliştiricisi vetarafından @wycats tarafından tasarlanmıştır

  • Knockout.js

    • Model-View-View Modeli (MVVM) uygulanarak dinamik JavaScript UI'larda kolaylaştırın.

  • Angular.js

    • Google tarafından HTML'in geleceği olarak tasarlanmıştır.

  • Cappuccino

    • Cappuccino masaüstü kalibreli uygulamalar oluşturmak için tasarlanan web tarayıcısında çalışan açıkkaynak kodlu bir yazılım omurgasıdır.

  • JavaScript MVC

    • JavaScriptMVC jQuery geliştirmede iyi fikirleri içeren bir açık kaynak kodlu bir yazılım omurgasıdır.

  • YUI

    • Yahoo! tarafından tasarlanan, YUI ücretsiz, zengin etkileşimli web uygulamaları oluşturmak için açık kaynak kodlu JavaScript ve CSS kütüphanesidir.

  • ZeptoJS

    • Zepto büyük ölçüde jQuery API uyumlu, modern tarayıcılar için minimalist bir JavaScript kütüphanesidir. Eğer jQuery kullanıyorsanız zaten Zeptoyu da nasıl kullanacağınızı biliyorsunuz.

  • Dojo Toolkit

    • Dojo yüksek performanslı web uygulamaları oluşturmak için ücretsiz, açık kaynak kodlu bir JavaScript kütüphanesidir. Proje sponsorları arasında IBM ve SitePen bulunmaktadır.

    OYUN MOTORLARI


  • MelonJS

    • MelonJS ücretsizdir, başlangıç için yeterli bir HTML5 oyun motorudur. Bu oyun motorunda tiled map formatı ile seviye tasarımını kolaylaştırabilirsiniz.

  • ImpactJS

    • ImpactJS 2010'un sonlarına doğru gelirken yayınlanmış bütün doğru ve test edilmiş HTMl5 oyun motorlarından biridir. ImpactJS çok iyi korunmuş ve güncellenmiş, ve destekleyen iyi bir topluluk vadır. ImpactJS bir çok döküman bulunmaktadır - hatta bu oyun motoru ile oyun geliştirmek hakkında iki tane kitap vardır.

  • LimeJS

    • LimeJS bütün masaüstü tarayıcıları ve modern dokunmatik ekranlar için native-experience oyunlarını, hızlı bir şekilde oluşturacağınız HTML5 kütüphanesidir.

  • Crafty

    • Crafty 2010'un sonlarında çıkmış bir oyun motorudur. Uzun bir süredir sıklıkla güncellenmektedir, ama son zamanlarda daha az güncellenmiştir. Tamamen güncellenmemiş demek doğru değildir buna rağmen hala geniş ölçüde kabul gören iyi bir HTML5 oyun motorudur.

  • Cocos2d-HTML5

    • Cocos2d-html5, MIT lisansı altında yayınlanmış açık kaynaklı web 2D kütüphanesidir. Cocos2d-x Projesinin HTML5 versiyonudur. Cocos2d-html5 geliştirme merkezinde native uygulamalar ve tarayıcılar arasında çapraz platform olan Cocos2d vardır.

  • Phaser

    • Phaser tamamen Flixel'e bağlıdır. Yıllardır HTML5 topluluğunda aktif olan Richard Davey (Photon Storm) tarafından oluşturulmuştur.

  • Goo

    • Goo tamamen webGL/HTML5 olarak tasarlanmış 3d JavaScript oyun motorudur.
